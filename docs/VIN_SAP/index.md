# [VIN_SAP](/docs/VIN_SAP)

## Docs
* Vin API : https://anypoint.mulesoft.com/exchange/portals/vingroup-4/

## API :

* Lấy danh sách logs : [vin/sap/logs](/vin/sap/logs)
* Lấy danh sách đơn hàng : [/vin/sap/orders](/vin/sap/orders)
* Lấy danh sách đơn bán (sale order) : [/vin/sap/orders](/vin/sap/saleorders)
* Lấy chi tiết đơn hàng theo mã đơn : [/vin/sap/orders/es_order_id](/vin/sap/orders/es_order_id)
